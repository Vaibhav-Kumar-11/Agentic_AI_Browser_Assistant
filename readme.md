# Agentic AI Browser Assistant

## What this project does
An AI-powered browser agent that autonomously fills forms, navigates websites, 
and completes web tasks using Python, Playwright, and LLM APIs.

## Week 1 - Environment Setup
- Python virtual environment with requests installed
- Async Python script that reads user info from JSON
- CSS selectors identified via Chrome DevTools
- Foundation for the memory layer (user_info.json)

## Tech Stack
- Python 3.10+
- Playwright (coming Week 2)
- FastAPI (coming Week 5)
- LLM APIs (coming Week 3)
